# 1950年 気象観測所（区内観測所）一覧データ

## 概要

1950年（昭和25年）時点の日本全国の気象観測所（区内観測所）の位置・設備情報をデジタル化したデータセットです。

## ファイル

- `japan_meteorological_stations_1950.csv` - 観測所データ（1,548件）

## 出典

中央氣象臺『区内観測所一覧表』昭和25年7月現在

https://dl.ndl.go.jp/pid/1374251

### デジタル化方法

Claude（Anthropic社のAI）を用いてPDFからOCRによりテキスト化し、その後手作業で校正・整形を行いました。

## 編集方針

### 原本からの変更点

1. **表記の統一**
   - 「〃」（同上記号）は全て実際の値に展開
   - 旧字体は原本のまま維持

2. **座標の追加**
   - 10進法緯度経度（lat_dd, lon_dd）を追加

3. **プライバシー配慮**
   - 観測場所が個人宅の場合、個人名を `[個人名]` に置換（53箇所）

4. **原典との差異**
   - 明らかな誤記と思われる箇所は修正し、remarks列に `[注:原典では〇〇と表記]` と記載

### 経緯度について

- 座標は日本測地系（旧測地系）です。
- 1分単位の座標のため、測地系のずれを含めて1〜2km程度の誤差があります。
- 一部の観測所（主に篤雨観測所）は原本に経緯度の記載がありません。これらは空欄のままとしています。
